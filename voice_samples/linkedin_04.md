# Type: Event recap + learnings post

Last Saturday, I got a chance to attend the n8n Automation Workshop + Buildathon, and it was such a fun and high-energy learning experience!

We got hands-on with building a RAG (Retrieval Augmented Generation) model using n8n, and the possibilities with these tools are mind-blowing once you dive in.
The session, led by Saurabh Khante, was packed with real-life examples, practical insights, and fresh perspectives on how we can use automation to simplify and speed up everyday tasks.

Here are some key takeaways:
1. The best way to learn these tools? Just start building. Learning by doing really works.
2. Automation vs Agent: Automations follow fixed steps and more rule based, while agents decide what to do based on goals and context.
3. RAG: Combines external data with LLMs to give more accurate, context-aware responses.
4. You don't need fancy infra to get started, tools like n8n let you build powerful workflows with little to no code.

We even built a basic version of a RAG bot. Sharing a snapshot of mine in the post!
